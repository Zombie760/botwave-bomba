# BotwaveBomba

**Not left/center/right. Five-axis bias fingerprints across Western, Adversarial, and Non-Aligned blocs. The gap IS the story.**

[![CI/CD](https://github.com/Zombie760/botwave-bomba/actions/workflows/botwavebomba.yml/badge.svg)](https://github.com/Zombie760/botwave-bomba/actions)

BotwaveBomba is a sovereign open-source news coverage platform built on the Botwave framework. It surfaces media bias, ownership trails, and coverage gaps across outlets and countries, with GitHub Pages as the public face and a Bun-backed backend API for dynamic data serving.

- **Live site:** https://zombie760.github.io/botwavebomba/
- **License:** MIT
- **Stack:** Bun/TypeScript backend, static frontend, GitHub Actions CI
- **Data:** `api/*.json` with source registry, ownership mappings, and clustered stories

## Project Structure

- `.github/workflows/` — CI/CD for GitHub Pages
- `api/` — static JSON data files
- `assets/` — CSS, JS, logos, manifest
- `.well-known/` — security contact
- `server/` — Bun backend for search/bootstrap
